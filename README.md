=========

This library implements a simple SMTP client.
See the [documentation](http://amousset.github.io/rust-smtp/smtp/) for more information.

Install
-------

To use this library, add the following to your `Cargo.toml`:

```toml
[dependencies]
smtp = "0.1.2"
```
